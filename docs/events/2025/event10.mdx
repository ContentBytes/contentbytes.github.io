# #10 | Let's workshop!

Content Bytes invites you to a hands-on workshop designed for technical writers who want to understand REST APIs by working with them directly.

APIs enable communication between applications — and understanding how to use them is key for anyone documenting software. This workshop is crafted specifically for technical writers who want to move beyond theory and gain practical API experience.

You'll learn how to:

​🔹 Send and read REST API requests

​🔹 Work with HTTP clients

​🔹 Handle authentication and headers

​🔹 Write a simple JavaScript script to send and process requests

This is a practical, learn-by-doing session — you'll spend most of your time experimenting, testing, and exploring how APIs behave in real-world use cases, supported by concise theory.

**Requirements**:

- Laptop with Wi-Fi
- Visual Studio Code installed
- „REST Client” extension by Huachao Mao installed in VS Code

**Tools**:
- https://restful-api.dev/

If you're a technical writer, API documentarian, or content creator looking to better understand the tools developers use every day — this workshop is for you.
